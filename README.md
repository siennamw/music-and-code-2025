# music-and-code-2025

This repo contains materials for the Music && Code course taught by [Sienna M. Wood](https://github.com/siennamw), block
3, Colorado College, 2025.

This includes JavaScript notes, instructions in Markdown,
and [several music encodings](#view-and-listen-to-mei-encodings) in [MEI](https://music-encoding.org/) format created
by the students in the course.

## View and Listen to MEI Encodings

Below are links to view the encodings contained in this repo via [mei-friend](https://mei-friend.mdw.ac.at/), an online
tool that provides a code viewer/editor, music notation rendering, and MIDI playback.

### Franz Danzi, Woodwind Quintet op. 56, no. 2 in G minor: I. Allegretto

This piece was encoded collaboratively by all class members. Each student was responsible for a single voice throughout.

[Franz Danzi, Woodwind Quintet op. 56, no. 2 in G minor: I. Allegretto]()

### Original Composition

This piece was composed and encoded collaboratively by all class members. The first contributor added 24 measures of a
single voice, the second contributor added a new voice with 24 measures of content, and so on. Once all contributions
had been made, the group collectively decided on shared characteristics of the piece such as dynamics.
